# Serp

Extract domains from google search engine results

## Installation

Add this line to your application's Gemfile:

    gem 'serp'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install serp

## Usage

    Serp.rank("keywords", 4) # search 4 pages of result for keywords
